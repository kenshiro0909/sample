HTML Customization Guide

はじめに：初心者の方へ

カスタマイズを行う場合は、Gemini（AI）に直接変更点を伝えて修正するようにしてください。

自身でコードを触るよりも、AIに指示を出す方が安全で確実です。

修正指示の3つのポイント

  • 変更したい箇所（どこ）
  • どのように修正したいのか
  • 指摘箇所以外は一切変更しないこと

基本テンプレート（コピペ用）

◯◯を△△に修正してください。 なお、指摘箇所以外は一切変更を加えないこと。

このような感じで、修正を一つ一つやっていく感じです✨️

エラー報告

Template

GitHub Pagesで公開したところ、以下の問題が発生しています。 ■ 問題の内容 （例：画像が表示されない / 崩れる / 動かない） ■ 現在のコード （問題がありそうな部分のスクショを貼り付け） 修正してください。

「何が」「どうなっているか」を具体的に伝えるのがポイントです。エラー画面のスクショを添えれば基本的に解決します。

その他の調整テクニック

記事の挿入

｛記事などの文章をそのままここに添付｝ 上記の文章をそのまま一切改変せずにサイトに最適な形で挿入してサイトを完成させて下さい。

インデックス回避 (noindex)

会員サイトなどで検索結果に表示させたくない場合

インデックスされたくないので、 ウェブページの HTML の <head> セクションに noindex メタタグを追加してください。 なお、指摘箇所以外は一切変更を加えないこと。
